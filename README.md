# AGTS: The Airborne Geophysics Tool Suite
Tools for common tasks in airborne science: mission planning, range calculcations, coordinate system transformations, etc. The main tool is a Jupyter notebook for B777 range estimates in the Arctic and Antarctic:

* **Jupyter:** [Notebook for B777 range estimates](https://github.com/mstudinger/airborne_tool_suite/blob/main/Python/B777/B777_range_Antarctica.ipynb) (requires external data files)
* **MATLAB®:** [Script for B777 range estimates](https://github.com/mstudinger/airborne_tool_suite/blob/main/MATLAB/b777_performance_antarctic.m) (requires external data files)
